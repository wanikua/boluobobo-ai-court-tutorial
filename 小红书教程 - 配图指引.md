## 🎨 截图工具推荐

### Windows
- **Snipaste**（推荐）：https://www.snipaste.com/
  - F1 截图，F3 贴图
  - 支持标注、箭头、文字
- **ShareX**：开源免费，功能强大

### Mac
- **系统自带**：Cmd+Shift+4
- **CleanShot X**：付费但好用

### Linux
- **Flameshot**：`sudo apt install flameshot`
- **系统自带**：PrtSc 键

### 在线工具
- **Excalidraw**：https://excalidraw.com/
  - 手绘风格流程图
  - 适合配图 7、配图 14

## 📝 截图注意事项

1. **敏感信息打码**
   - Token/API Key 全部打码
   - 邮箱地址部分打码
   - 服务器 IP 可以保留（已公开）

2. **统一风格**
   - 所有截图用相同的标注颜色（推荐红色）
   - 箭头统一用红色或黄色
   - 文字说明用白色背景黑字

3. **尺寸建议**
   - 宽度：1080px（适配小红书）
   - 高度：根据内容自适应
   - 格式：PNG（清晰）或 JPEG（体积小）

4. **标注规范**
   - 🔴 红圈：重点按钮/输入框
   - ⬇️ 箭头：操作流程
   - 📝 文字：简短说明
