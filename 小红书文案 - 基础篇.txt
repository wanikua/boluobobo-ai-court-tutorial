🚀 3 步搭建（30 分钟搞定）

第 1 步：领免费服务器（10 分钟）
去 Oracle Cloud 注册账号，领一台永久免费的云服务器：
• 配置：4 核 24GB 内存 + 200GB 硬盘 + 10TB 流量
• 价值：≈$100/月，但 Oracle 免费送！

⚠️ 关键点：
• 选日本东京区域（容量足）
• 一定要选 ARM 架构（Ampere）
• 下载好 SSH 密钥（只此一次！）

❗ 易错点：信用卡验证用 Visa/MasterCard，银联不行！

━━━━━━━━━━━━━━━━━━━━

第 2 步：一键安装 Clawdbot（5 分钟）
SSH 连上服务器，复制粘贴：

# 1. 安装 Node.js
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.3/install.sh | bash
source ~/.bashrc
nvm install 22

# 2. 安装 Clawdbot
npm install -g clawdbot

# 3. 初始化工作目录
mkdir -p ~/clawd && cd ~/clawd
clawdbot configure

# 4.（可选）安装 Claude Code
npm install -g @anthropic-ai/claude-code

✅ 成功标志：运行 clawdbot --version 看到版本号就对了！

━━━━━━━━━━━━━━━━━━━━

第 3 步：配置朝廷架构（15 分钟）

1️⃣ 获取 API Key（3 分钟）
• 访问 https://console.anthropic.com
• 注册/登录 → API Keys → Create Key
• 复制 Key（格式：sk-ant-api03-xxxxx）

🔴 重要：Key 只显示一次！立刻复制保存！

2️⃣ 配置环境变量（1 分钟）
echo 'export ANTHROPIC_API_KEY="sk-ant-api03-你的密钥"' >> ~/.bashrc
source ~/.bashrc

3️⃣ 创建朝廷配置文件（10 分钟）
在服务器创建 ~/clawd/IDENTITY.md 和配置文件

4️⃣ 启动朝廷（1 分钟）
clawdbot gateway start
clawdbot gateway status

✅ 成功标志：看到"active (running)"就对了！

❓ 常见问题 Q&A

Q: Oracle 注册一直失败怎么办？
A: 检查是否用了国内 IP（换梯子节点），检查信用卡类型（必须 Visa/MasterCard），换个邮箱试试（Gmail 最稳）

Q: 显示"Out of capacity"怎么办？
A: 换其他区域（ap-tokyo-1 / ap-seoul-1），换时间段（凌晨美国时间成功率高），等几天再试
